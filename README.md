# Задача А1. Задача трех кругов.

## Файлы с данными

### area_data.csv
Данные для построения графика зависимости вычисленной приближенной площади от количества точек.

**Структура данных:**
- **N** - количество сгенерированных точек (диапазон: 100-100000)
- **ScaleFactor** - масштаб области генерации:
  - 0.5 - узкая область
  - 1.0 - нормальная область  
  - 2.0 - широкая область
- **ComputedArea** - вычисленная приближенная площадь пересечения окружностей

### error_data.csv
Данные для построения графика относительного отклонения вычислений.

**Структура данных:**
- **N** - количество сгенерированных точек (диапазон: 100-100000)
- **ScaleFactor** - масштаб области генерации (0.5, 1.0, 2.0)
- **RelativeError** - относительная ошибка от точного значения площади (0.4438)

## Использование данных

Данные предназначены для анализа влияния параметров алгоритма на точность вычислений:
- Зависимость точности от количества точек
- Влияние масштаба области генерации на эффективность метода
  
Каждая строка в файлах представляет результат одного эксперимента с определенной комбинацией параметров N и ScaleFactor.
